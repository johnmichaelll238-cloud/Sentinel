# Sentinel

Sentinel is an intelligent infrastructure monitoring platform that collects, stores, analyzes, and eventually predicts system behavior.

## Current Features

- CPU telemetry collection
- SQLite storage
- Modular collector architecture

## Roadmap

- RAM monitoring
- Disk monitoring
- Network monitoring
- FastAPI service
- Streamlit dashboard
- Docker deployment
- Forecasting
- Anomaly detection

## Tech Stack

- Python
- SQLite
- psutil
- FastAPI
- Streamlit
- Docker (planned)
